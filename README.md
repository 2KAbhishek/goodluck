## Prerequisites

Before you begin, ensure you have met the following requirements:

- You have installed the latest version of `go` & `fortune`.

## Installing goodluck

To install goodluck, follow these steps:

```bash
git clone https://github.com/2kabhishek/goodluck
cd goodluck
go build
go install
```

## Using goodluck

```bash
USAGE:
    goodluck

```
